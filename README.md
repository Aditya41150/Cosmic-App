## Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK (>=2.17.0)
- Android Studio or Xcode

## Installation

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Building for Production

```bash
flutter build apk      # Android
flutter build ios      # iOS
flutter build web      # Web
```
